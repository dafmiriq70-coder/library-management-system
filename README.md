# PHP Library Management System

![Library Image](https://via.placeholder.com/800x200.png?text=Library+Management+System)

A simple **Library Management System** built with **PHP, MySQL, HTML, CSS, and JavaScript** to manage books, users, and library operations efficiently.

---

## Features

- User registration and authentication  
- Book catalog management  
- Borrowing and returning books  
- Fine management for overdue books  
- User profile management  
- Reports and statistics for library operations
